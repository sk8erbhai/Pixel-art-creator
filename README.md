PixelArtCreator is a free online pixel art maker that allows users to draw pixel art
by tracing a reference image on a pixel grid.

## Features
- Draw pixel art manually on a grid
- Upload and use reference images
- Adjustable grid size
- Beginner-friendly pixel art editor
- No login required

## Use Cases
- Game sprite creation
- Pixel icons and avatars
- Learning pixel art by tracing
- Retro-style art
